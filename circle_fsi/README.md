# circle_fsi — 圆形刚体 FSI Demo

本目录将刚体形状从**矩形**改为**圆形**，演示溃坝水流冲击圆球并使其漂浮 / 旋转的效果。圆形的边界粒子采样使用极坐标均匀分布，流体求解器与 `taichi_parallel` 保持一致。

> **注意：** 本版本已移除刚体间碰撞（`CollisionSolver`），多个圆球之间会穿透。

## 特点

- 刚体形状：圆形（半径可在 `config.py` 中调整）
- 流体求解：WCSPH（与 `taichi_parallel` 相同）
- 无刚体间碰撞（简化版本，圆球间不检测 / 响应碰撞）
- 渲染：圆形轮廓（多边形近似，36 段）+ 旋转指针

## 文件说明

| 文件 | 职责 |
|------|------|
| `main.py` | 入口：窗口、事件循环、渲染 |
| `config.py` | 全局参数（圆形专属：`BODY_RADIUS`、`N_CIRCLE_SEGS`） |
| `particles.py` | 流体 / 墙壁 / 圆形边界粒子生成 |
| `simulation.py` | Taichi FSI 引擎（圆形刚体积分，无碰撞模块） |

## 运行

```bash
cd SPH/fsi/circle_fsi
python main.py
```

## 交互

| 操作 | 效果 |
|------|------|
| 左键点击 | 在鼠标位置投入新圆球（最多 10 个） |
| 关闭窗口 | 退出模拟 |

## 颜色说明

| 颜色 | 含义 |
|------|------|
| 深蓝/青渐变 | 流体（速度越大越亮） |
| 灰色 | 墙壁边界粒子 |
| 彩色轮廓线 | 圆形刚体（每个颜色不同） |

## 主要参数（config.py）

| 参数 | 说明 | 默认 |
|------|------|------|
| `BODY_RADIUS` | 圆球半径 | 0.12 m |
| `BODY_DENSITY` | 密度（< 1000 会浮） | 500 kg/m³ |
| `N_CIRCLE_SEGS` | 轮廓多边形段数 | 36 |
| `PARTICLE_RADIUS` | 流体粒子半径 | 0.005 m |
| `VISCOSITY` | 粘性系数 | 0.015 |

## 与矩形版本的算法差异

| 方面 | 矩形（taichi_parallel） | 圆形（circle_fsi） |
|------|------------------------|-------------------|
| 边界粒子采样 | 矩形边均匀采样 | 极坐标均匀采样 |
| 转动惯量 | `I = m(W²+H²)/12` | `I = ½mr²` |
| 刚体间碰撞 | SAT + Sequential Impulse | 无 |
| 渲染 | 矩形边框（4 条线段） | 多边形近似圆（36 段） |
